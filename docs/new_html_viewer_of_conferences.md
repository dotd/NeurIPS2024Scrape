In this file we define how to build an html that can filter the list of all papers which are referenced in ConferenceTables. 

On the top of the html we will have some predefined filters, expressed by buttons. 

The buttons for the topics are:
- Robotics
- Reinforcement Learning
- VLA
- VLM
- Open Models
- World Models

The buttons for the conferences are all the conferences (which can be identified by the name of the csv)

The list of papers will be drop down lines in the html.
When the drop down of a paper is closed, onlt the name of the paper together with some small icon that identifies the topic of the paper.
The small icons should be similar to the topics mentioned above.

There will be also some search bar that free words can be entered and added as well.

the name of the file should be htmls/all_conferences_filter.html
